# Eenvoudige Contactenlijst Manager

Ontwikkel een eenvoudige applicatie die het mogelijk maakt om contacten toe te voegen, te bekijken en te verwijderen. De applicatie moet de contactgegevens opslaan in een lijst binnen het geheugen. Dit project is bedoeld om basiskennis van objectgeoriënteerd programmeren en dataopslag in het geheugen te toetsen.

## Opdracht

Implementeer de gevraagde functionaliteit.
